# 🏠 بوت حمد نت - Hamad Net Bot

بوت تليجرام متكامل لإدارة ومراقبة شبكة **حمد نت**. يوفر التحكم الكامل بالشبكة، كشف الاختراقات، مراقبة الإنترنت، والتشخيص التلقائي للمشاكل.

## ✨ المميزات

### 📱 مراقبة الأجهزة
- كشف جميع الأجهزة المتصلة بالشبكة
- تنبيه فوري عند دخول أو خروج أي جهاز
- تحديد الأجهزة المرخصة وغير المرخصة
- تسمية الأجهزة بأسماء مستعارة
- كشف نوع الجهاز والشركة المصنعة

### 🔒 الأمان والحماية
- كشف محاولات الاختراق والهجمات
- كشف هجمات القوة الغاشمة (Brute Force)
- كشف فحص المنافذ (Port Scanning)
- كشف انتحال MAC (MAC Spoofing)
- حظر تلقائي للتهديدات (اختياري)
- فحص الثغرات الأمنية في الراوتر

### 📡 مراقبة الإنترنت
- كشف انقطاع الإنترنت فورياً
- تشخيص سبب الانقطاع تلقائياً
- تحديد المنطقة المتأثرة
- قياس التأخير (Latency)
- فحص DNS و HTTP
- دعم Starlink
- تتبع المسار (Traceroute)

### 📊 الإحصائيات والمراقبة
- مراقبة الباندويث لكل واجهة
- أكبر مستخدمي الباندويث
- تحديد سرعة الأجهزة (Rate Limiting)
- مراقبة استخدام المعالج والذاكرة

### ⚙️ التحكم عن بُعد
- حظر/إلغاء حظر الأجهزة
- ترخيص/إلغاء ترخيص الأجهزة
- تحديد سرعة التحميل والرفع
- إعادة تشغيل الراوتر
- فحص الشبكة يدوياً
- عرض سجلات النظام

### 🔔 التنبيهات الفورية
- جهاز جديد ينضم للشبكة
- جهاز يغادر الشبكة
- محاولة اختراق
- انقطاع الإنترنت واستعادته
- تغيير في بنية الشبكة (مودم/سويتش جديد)
- استخدام باندويث مرتفع

## 🖥️ الراوترات المدعومة

| الراوتر | طريقة الاتصال | الميزات |
|---------|-------------|---------|
| MikroTik RouterOS | API (المنفذ 8728) | كاملة |
| OpenWrt | SSH | كاملة |

## 📋 المتطلبات

- Python 3.8 أو أحدث
- تليجرام بوت توكن (من @BotFather)
- راوتر MikroTik أو OpenWrt
- (اختياري) nmap لفحص أفضل
- (اختياري) traceroute لتتبع المسار

## 🚀 التثبيت

### 1. استنساخ المشروع
```bash
git clone https://github.com/YOUR_USERNAME/hamad-net-bot.git
cd hamad-net-bot
```

### 2. تشغيل سكربت التثبيت
```bash
chmod +x install.sh
./install.sh
```

### 3. إعداد البوت
```bash
cp .env.example .env
nano .env
```

**يجب تعديل الإعدادات التالية:**

| الإعداد | الوصف |
|---------|-------|
| `TELEGRAM_BOT_TOKEN` | توكن البوت من @BotFather |
| `AUTHORIZED_CHAT_IDS` | معرف محادثتك في تليجرام |
| `ROUTER_TYPE` | نوع الراوتر: `mikrotik` أو `openwrt` |
| `MIKROTIK_HOST` | عنوان IP الراوتر |
| `MIKROTIK_PASSWORD` | كلمة مرور الراوتر |

#### كيفية الحصول على Chat ID:
1. أرسل رسالة لبوت @userinfobot
2. سيقوم بإرسال معرف المحادثة الخاص بك
3. أضفه في `AUTHORIZED_CHAT_IDS`

### 4. تشغيل البوت
```bash
source venv/bin/activate
python bot.py
```

### 5. (اختياري) تشغيل كخدمة نظام
```bash
sudo cp scripts/hamad-net-bot.service /etc/systemd/system/
sudo systemctl enable hamad-net-bot
sudo systemctl start hamad-net-bot
```

## 🛰️ دعم Starlink

لتفعيل مراقبة Starlink:
1. فعّل `STARLINK_ENABLED=true` في `.env`
2. حدد عنوان IP الخاص بـ Dishy: `STARLINK_IP=192.168.100.1`
3. (اختياري) ثبّت `grpcurl` لقراءة بيانات Dishy التفصيلية

## 🔧 إعدادات MikroTik

### تفعيل API
```
/ip service enable api
/ip service set api port=8728 address=192.168.1.0/24
```

### تأمين API
```
/ip firewall filter add chain=input protocol=tcp dst-port=8728 src-address=192.168.1.0/24 action=accept
/ip firewall filter add chain=input protocol=tcp dst-port=8728 action=drop
```

## ⚠️ تنبيهات أمنية

- لا تشارك ملف `.env` مع أي شخص
- استخدم كلمات مرور قوية للراوتر
- قيّد وصول API على الشبكة المحلية فقط
- فعّل التشفير على اتصال الراوتر
